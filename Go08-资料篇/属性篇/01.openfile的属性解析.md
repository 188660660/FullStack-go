os.OpenFile(文件路径，打开模式，权限)用来打开文件

**模式：**

| 常量     | 描述 |
| -------- | ---- |
| O_RDONLY | 只读 |
| O_WRONLY | 只写 |
| O_RDWR   | 读写 |
| O_APPEND | 追加 |

**权限**

| 常量 | 描述                                   |
| ---- | -------------------------------------- |
| 0    | 没有任何权限                           |
| 1    | 可执行（如果是可执行文件，就可以运行） |
| 2    | 写                                     |
| 3    | 写、可执行                             |
| 4    | 读                                     |
| 5    | 读、可执行                             |
| 6    | 读、写                                 |
| 7    | 读、写、可执行                         |
